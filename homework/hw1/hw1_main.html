Machine learning is the ability for a computer to perform additional tasks such as identification of new categories without additional programming.
That means that the original code of the application is dynamic enough to "learn" how to do a set of specified tasks on its own!

Some of the techniques growing in the field include deep learning, pattern recognition and computational learning!
