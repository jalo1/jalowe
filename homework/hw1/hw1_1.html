Deep learning focuses on using what clusters of data represent instead of using specific task focused algorithms.

Current models are trying to recreate the way neural networks inside the brain work in order to get a better picture of
what data representation looks like to us. If we can figure out more details about our own brains, it could lead to making
better machine brains!

"Training Programs" run a series of data points through the model in an attempt to flesh out what makes certain elements represent specific concepts.
